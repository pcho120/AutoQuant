## Implementation Details

### 1. NewsAPI Integration (fetch_news)
- Endpoint: https://newsapi.org/v2/everything
- Query parameters: ticker (q), date range (from/to), language=en, sortBy=relevancy
- Date range: Uses datetime.timedelta to calculate last N days
- Timeout: 10 seconds
- Error handling: All exceptions caught, returns empty list []
- Response parsing: Extracts "articles" array from JSON response

### 2. Sentiment Analysis (get_sentiment)
- Method: Keyword-based heuristic (fast, no ML dependencies)
- Positive keywords (10): profit, growth, bullish, upgrade, beat, surge, gains, strong, outperform, rally
- Negative keywords (10): loss, decline, bearish, downgrade, miss, plunge, weak, underperform, sell, crash
- Text processing: Combines title + description, converts to lowercase
- Formula: (pos_count - neg_count) / (pos_count + neg_count)
- Bounding: max(-1.0, min(1.0, raw_sentiment)) ensures [-1.0, 1.0] range
- Neutral case: Returns 0.0 when no keywords found

### 3. API Key Management
- Constructor parameter: api_key (str)
- Supports st.secrets pattern: st.secrets.get('newsapi', {}).get('key', 'DEMO_KEY')
- Supports direct injection: NewsProvider('my-api-key')
- No hardcoded keys in source code
